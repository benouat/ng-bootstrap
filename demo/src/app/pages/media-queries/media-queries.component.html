Media Observer is an utility service. It provides methods to verify and/or observe changes occuring on specific media queries.

Bootstrap already defines its own responsive breakpoints. For you to seamlessly use them, we provide an injection token exposing the below exposed media queries.

These breakpoints values are dynamically computed at runtime based on Bootstrap CSS variables.
It means that if you have your own custom SCSS Bootstrap theme and you override these default values, or if you add new ones, it will work just fine with your new values.

BOOTSTRAP_BREAKPOINTS token can be injected along with NgbMediaObserver service.

We also provide a special *ngIf directive via *ngIfMedia. For an improved DX, all BOOTSTRAP_BREAKPOINTS media-queries can be used implicitely.
Providing any of 'xs', 'sm', 'md', 'lg' or 'xl' will be converted internally to the corresponding media-query.
